The Box Model

Every element in CSS is a rectangular box. Understanding its layers is the key to mastering layout.

Anatomy of a Box

margin
border
padding
content

Explore Each Layer

🔲

Margin

The outermost layer — transparent space that separates the element from its neighbors. It collapses vertically and can even be negative.

margin: 20px auto;
🖼️

Border

A visible boundary wrapping the padding and content. It can be styled with different widths, colors, and patterns like solid, dashed, or double.

border: 3px solid #a78bfa;
🧱

Padding

Breathing room between the content and the border. Unlike margin, padding inherits the element's background color, giving content visual comfort.

padding: 16px 24px;
✏️

Content

The innermost area where your actual text, images, and child elements live. Its size is controlled by width and height properties.

width: 300px; height: auto;

💡 Pro Tip: box-sizing: border-box

By default, width and height only set the content size. Padding and border are added on top, making elements larger than expected. Use box-sizing: border-box to include padding and border within the declared width.

*, *::before, *::after {
  box-sizing: border-box;
}
